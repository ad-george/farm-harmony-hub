# Farm Management System

A comprehensive farm management system built with Flask, MySQL, and modern web technologies.

## Features
- **User Management**: Role-based access control (Admin, Manager, Staff)
- **Farm Management**: Multiple farm tracking with detailed information
- **Crop Management**: Planting to harvest tracking with growth stages
- **Livestock Management**: Animal tracking with health monitoring
- **Financial Management**: Income/expense tracking with analytics
- **Task Management**: Assign and track farm tasks
- **Reporting**: Generate detailed reports and analytics
- **Dashboard**: Real-time data visualization

## Tech Stack
- **Backend**: Python Flask
- **Database**: MySQL with SQLAlchemy ORM
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5
- **Authentication**: Flask-Login with bcrypt hashing
- **Forms**: Flask-WTF with CSRF protection
